# SQL + Power BI Sales Analytics Dashboard

An interactive Business Intelligence dashboard built using SQL and Power BI to analyze sales performance, profitability, customer behavior, product trends, and regional insights.

---

# Project Overview

The dataset was first analyzed using SQL for querying and business analysis before building interactive dashboards in Power BI.

This project focuses on transforming raw sales data into actionable business insights through data cleaning, SQL analysis, and interactive Power BI visualizations.

The dashboard helps analyze:
- Sales performance
- Profitability trends
- Customer insights
- Regional performance
- Product analysis
- Shipping trends

---

---

# Key Business Insights

- Technology category generated the highest profit.
- Furniture category showed weaker profitability despite high sales.
- High discounts negatively impacted profit margins.
- West region emerged as the best-performing region.
- Some cities generated consistent losses and require optimization.
- Consumer segment contributed the majority of sales.
- Sales peaked during the final quarter of the year.
